# DSA Java

A comprehensive collection of **Data Structures and Algorithms** problems solved in Java (with a few solutions in Python and C).

---

## 📁 Folder Structure

```
DSA_java/
├── Array/               # Array manipulation, searching, sliding window
│   └── Sorting/         # Sorting algorithms & sorting-related problems
├── Basic/               # Foundational math & number theory
│   └── Bit_Magic/       # Bitwise operation problems
├── Leetcode/            # LeetCode solutions organized by topic
│   ├── Arrays/          # Array manipulation and subarray problems
│   ├── BinarySearch/    # Binary search problems
│   ├── BitManipulation/ # Bit manipulation problems
│   ├── Math/            # Math & number theory problems
│   ├── Matrix/          # 2D array and matrix problems
│   ├── Stack/           # Stack-based problems
│   ├── Strings/         # String manipulation problems
│   ├── Trees/           # Binary tree & BST problems
│   └── L75/             # LeetCode 75 Study Plan solutions
├── Linked_List/         # Linked list problems
├── Recursion/           # Recursive algorithm problems
├── Stack/               # Stack data structure problems
├── Strings/             # String manipulation problems
└── Trees/               # Tree data structure problems
```

---

## 📂 Topic Breakdown

### Array
Array manipulation, searching, two-pointer, prefix sum, and sliding window problems from GeeksForGeeks and similar platforms.

| Sub-folder | Contents |
|---|---|
| `Array/` | TwoSum, ThreeSum, peak element, rotation, equilibrium point, majority element, subarray sums, etc. |
| `Array/Sorting/` | Bubble Sort, Insertion Sort, Quick Sort, Counting Sort, merge problems, inversions, Kth smallest, etc. |

---

### Basic
Foundational math and number theory problems.

| Sub-folder | Contents |
|---|---|
| `Basic/` | Factorial, GCD/LCM, palindrome, primality, Armstrong numbers, digit operations, etc. |
| `Basic/Bit_Magic/` | Set bits, bit difference, power of 2, divide using bits, etc. |

---

| Sub-folder | Contents |

## 🛠 How to Run

Each file is a standalone Java class. Compile and run individually:

```bash
javac Array/TwoSum.java
java Array/TwoSum
```

## 📌 Notes

- Primary language: **Java**
- A few solutions are also available in **Python** (`.py`) and **C** (`.c`)
- LeetCode problems are named with their problem number prefix: `704_BinarySearch.java`
